## Why divmemory?

<CardGrid stack>
	<Card title="Zero Git Noise" icon="error">
		No temp files, no git commits, no repository pollution. Memories are stored securely in Cloudflare D1.
	</Card>
	<Card title="Instant Sync" icon="rocket">
		Injects cached context into the agent at session start, and asynchronously saves facts at session end.
	</Card>
	<Card title="Cross-Session Persistence" icon="document">
		Maintains memory across multiple chat sessions, allowing agents to build on top of past context.
	</Card>
	<Card title="Highly Extensible" icon="setting">
		Works natively as a Factory Droid plugin and a simple REST API.
	</Card>
</CardGrid>
