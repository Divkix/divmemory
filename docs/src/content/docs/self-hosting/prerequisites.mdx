---
title: Prerequisites
description: Everything you need before deploying divmemory.
---

- [Bun](https://bun.sh) >= 1.3
- [Wrangler](https://developers.cloudflare.com/workers/wrangler/) CLI — installed via this repo or via `npx wrangler`
- Node.js 20+ for Droid hook scripts
- Cloudflare account with [Workers](https://workers.cloudflare.com) and [D1](https://developers.cloudflare.com/d1) enabled
- Fireworks AI account with [Firepass](https://fireworks.ai) subscription
- Droid/Factory CLI with plugin support
