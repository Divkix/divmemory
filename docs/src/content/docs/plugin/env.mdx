---
title: Environment Variables
description: All environment variables for the plugin and CLI.
---

| Variable | Required | Where | Default | Description |
| --- | --- | --- | --- | --- |
| `DIVMEMORY_API_KEY` | Yes | Shell / Droid env | — | Auth token for Worker API. Same value on every laptop. |
| `DIVMEMORY_WORKER_URL` | Yes | Shell / Droid env | — | Your deployed Worker URL. |
| `DIVMEMORY_HOME` | No | Shell / Droid env | `~/.divmemory` | Local cache/queue directory. |

:::caution[Security Warning]
`DIVMEMORY_API_KEY` must be treated as a secret and never committed to source control (e.g., do not check in `.env`, `.dev.vars`, or similar files). Use the documented [secrets workflow](/self-hosting/secrets) (e.g., `wrangler secret` or your repository's secrets management) for storing and setting the key in production and local environments.
:::
