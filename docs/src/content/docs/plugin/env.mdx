---
title: Environment Variables
description: All environment variables for the plugin and CLI.
---

| Variable | Required | Where | Default | Description |
| --- | --- | --- | --- | --- |
| `DIVMEMORY_API_KEY` | Yes | Shell / Droid env | — | Auth token for Worker API. Same value on every laptop. |
| `DIVMEMORY_WORKER_URL` | Yes | Shell / Droid env | — | Your deployed Worker URL. |
| `DIVMEMORY_HOME` | No | Shell / Droid env | `~/.divmemory` | Local cache/queue directory. |
